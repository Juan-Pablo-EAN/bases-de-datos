MySQL Analytics Engine: Oracle dota de inteligencia a las BBDD

15 septiembre, 2021

Oracle acaba de presentar Oracle MySQL Database Service con motor de análisis MySQL, un servicio optimizado y disponible exclusivamente en Oracle Cloud Infrastructure (OCI). Se trata del único servicio MySQL que ofrece a los clientes una plataforma individual y unificada tanto para cargas de trabajo Online Transactional Processing (OLTP) como para Online Analytics Processing (OLAP) para crear y ejecutar aplicaciones modernas de manera más rápida y segura. MySQL Analytics Engine, desarrollado para MySQL Database Service por el equipo de ingeniería MySQL de Oracle en colaboración con Oracle Labs, es un nuevo acelerador analítico in-memory que escala a miles de núcleos, admite análisis en tiempo real y es hasta 17 veces más rápido y un 66 por ciento menos costoso que Amazon Redshift.

MySQL es reconocida como la base de datos más popular del planeta1 y es utilizada por las redes sociales, empresas de comercio electrónico y bancos más grandes del mundo, así como por los fabricantes y empresas de alta tecnología más innovadores. Si bien las bases de datos MySQL se usan comúnmente para OLTP, la mayoría de las empresas que ejecutan bases de datos MySQL también necesitan ejecutar análisis de datos almacenados en bases de datos MySQL. Hasta hoy, las organizaciones tenían que extraer, transformar y cargar (ETL) sus datos en una base de datos de análisis separada, un proceso costoso y complejo que reduce el rendimiento, la seguridad y la calidad de los datos, retrasando la implementación de nuevas aplicaciones.

El servicio de base de datos Oracle MySQL con MySQL Analytics Engine proporciona un servicio de escalamiento horizontal completamente gestionado para análisis y OLTP y es el único servicio que permite a los clientes ejecutar cargas de trabajo OLTP y OLAP directamente desde su base de datos MySQL. Cuando los usuarios ponen en marcha el servicio, los datos de la BBDD MySQL se completan en la memoria del motor de análisis, eliminando la necesidad de transferir datos ETL a bases de datos especializadas. El servicio funciona perfectamemente con cualquier herramienta y aplicación compatible con MySQL y enrutará automáticamente las consultas al motor de análisis de MySQL, mejorando drásticamente el rendimiento. Cualquier cambio en la BBDD MySQL se completa automáticamente en el motor de análisis en tiempo real. Esto simplifica el desarrollo de aplicaciones, ahorra tiempo, permite consultas en tiempo real, reduce los costes y mejora la seguridad. El nuevo servicio está disponible en todas las regiones comerciales de Oracle Cloud y lo estará en Oracle Dedicated Region Cloud@Customer.

“MySQL es la base de datos más popular entre los desarrolladores y es ampliamente utilizada por empresas de todos los sectores. Pero, hasta el día de hoy, los usuarios de MySQL se han visto obligados a mover sus datos a almacenes de datos incompatibles separados para análisis, lo que genera costes más altos y demora la respuesta”, dijo Edward Screven, chief corporate architect, Oracle. “Con las últimas innovaciones del equipo de ingeniería de MySQL, Oracle es el único proveedor que ofrece a los desarrolladores y gestores de bases de datos una plataforma individual y unificada que puede ejecutar fácilmente análisis de alto rendimiento en tiempo real en su base de datos MySQL sin requerir ningún cambio en sus aplicaciones MySQL, lo que hace que Oracle Cloud Infrastructure sea el mejor lugar para ejecutar aplicaciones MySQL”.

MySQL Database Service es un servicio de base de datos totalmente gestionado que permite a los clientes aprovisionar bases de datos MySQL en minutos y concentrarse en el desarrollo, sin tener que preocuparse por las tareas de gestión de bases de datos que consumen mucho tiempo -como el escalado, copia de seguridad, parches del sistema operativo, mantenimiento del servidor-. El servicio de BBDD MySQL es el único de nube pública creado en MySQL Enterprise Edition para ayudar a garantizar la máxima fiabilidad y seguridad. El nuevo servicio es perfectamente compatible con MySQL on-premises, lo que facilita a los clientes mover aplicaciones y cargas de trabajo MySQL existentes a Oracle Cloud. Como resultado, los desarrolladores siempre tienen acceso a las últimas funciones de MySQL y pueden aprovechar los servicios nativos en OCI utilizando tecnologías como Kubernetes y Docker.

MySQL Analytics Engine está diseñado para una escala y un rendimiento masivos, lo que acelera 400 veces el rendimiento de MySQL para las consultas analíticas. En comparación con Amazon Aurora, el servicio de base de datos MySQL con MySQL Analytics Engine integrado es más de 1.a000 veces más rápido y a un tercio del coste de las cargas de trabajo analíticas como TPC-H. El motor de análisis ofrece procesamiento de columnas híbrido in- memory, paralelismo masivo entre e intra nodos optimizado para Oracle Cloud Infrastructure y algoritmos de procesamiento de consultas distribuidas, al tiempo que libera a los DBA de las tareas ETL que consumen mucho tiempo.
