Contextualización de bases de datos

¿Qué es un motor de bases de datos?

Los motores de bases de datos son programas intermedios entre las bases de datos y la aplicación que usa la base de datos, cada motor se encarga de una tarea en especifica como crear la base de datos o administrar el acceso a esta. Los motores de bases de datos optimizan y agilizan el proceso de acceso, consulta y extracción de datos en un software, estos deben cumplir con las siguientes caracteristicas:

  • Abstracción de la información
  • Independencia
  • Redundancia mínima
  • Consistencia
  • Seguridad Integridad
  • Respaldo y recuperación
  • Control de la concurrencia
  • Tiempo corto de respuesta.

Los motores de bases de datos son sistemas internos que ayudan de manera óptima a la creación de las bases de datos, estos estan integrados en los gestores de bases de datos que son el software que nos facilita conectarnos con el motor de bases de datos. Entre los gestores de bases de datos más conocidos están MySQL Server, SQL Server, Oracle, Microsoft Access, entre otros.

¿Qué características tiene una base de datos relacional?

Las bases de datos relacionales son las más usadas actualmente en todo el mundo, también son llamadas bases de datos SQL, estas almacenan datos relacionados entre sí, como su mismo nombre lo dice, donde los datos son representados sin necesidad de jerarquizarlos.

Las bases de datos relacionales cuentan con las siguientes características:

  • Son u conjunto de tablas formadas por filas donde cada fila debe tener un identificador clave (ID)
  • Los ID facilitan el acceso a la información
  • Se evitan la duplicidad de registros
  • Si un registro es eliminado no deben verse afectados los restantes
  • No pueden existir dos tablas con el mismo nombre

Ventajas de usar bases de datos SQL

  • Permite el manejo de grandes cantidades de datos de manera segura.
  • Garantizan la prevención de duplicidad de registro.s
  • Se puede bloquear el acceso a ciertos datos mientras estos se actualizan.
  • Permite el consumo de datos de varios usuarios al mismo tiempo a traves de la concurrencia.

Desventajas de usar bases de datos SQL

  • Deficientes a la hora de manejar datos gráficos o multimedia.
  • No permiten crear tablas de forma jerárquica
